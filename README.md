# ⚡ Alpha Quant System
### Système de Trading Algorithmique pour le NASDAQ 100.

Ce projet utilise Python et Streamlit pour centraliser les alertes de trading et analyser la pertinence des signaux via un moteur de probabilités.

**Fonctionnalités :**
- 🛰️ Réception de Webhooks TradingView (Signaux Alpha/Fusion).
- 🧠 Calcul de probabilités de succès (Moteur Quant).
- 📊 Journal de bord automatisé.
- 🛡️ Gestion du risque (1 contrat fixe).
